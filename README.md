# zenodo-dl

A simple CLI to download all files from a Zenodo record to a target folder.


How to use:

```sh
git clone git@github.com:swenkel/zenodo-dl.git
cd zenodo-dl
cargo run --release -- -r 10700792 -o /tmp/dcase_2024_task_10/
```
